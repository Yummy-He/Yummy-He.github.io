# 简介
基于 AI Agent 的中国公募基金智能分析系统，通过历史数据回测与强化学习，让 AI 在多轮投资模拟中积累经验并生成投资建议。

主要功能包括基金数据采集、AI 回测、多轮学习、投资建议生成及定期报告产出。
